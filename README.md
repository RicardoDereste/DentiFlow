**DentiFlow** is a modern, high-performance dental clinic website built with **React**. It serves as a comprehensive digital storefront for a professional clinic, offering an intuitive user experience across five key pages: Home, Services, About Us, FAQ, and Contact.

The primary focus of this build was to balance **aesthetic professional design** with **technical reliability**. This involved creating a custom, accessible mobile navigation system, fine-tuning responsive CSS grids for clinical service showcases, and integrating a functional Google Maps and contact system.

## âœ¨ Features

- **Modern Navigation System:** A custom-built, mobile-optimized header featuring a hamburger menu that remains as three lines and changes color when active, providing a seamless transition between mobile and desktop views.
- **Service Showcases:** Dedicated, image-rich sections for General, Cosmetic, and Restorative dentistry using responsive CSS grids.
- **Dynamic FAQ Accordion:** A custom-coded "drop-down" system for frequently asked questions, utilizing **Font Awesome** icons for an intuitive toggle experience.
- **Font Awesome 7.0.1 Integration:** High-fidelity, scalable iconography used throughout the site for USPs (Unique Selling Propositions), service lists, and contact details.
- **Responsive Contact Hub:** A specialized contact page featuring a functional inquiry form and a real embedded Google Maps location for patient convenience.
- **Consistent Design System:** Implementation of a cohesive color palette managed via CSS Variables:
  - **Primary Blue:** Trust and professionalism.
  - **Secondary Teal:** Cleanliness and health.
  - **Accent Orange:** High-visibility Calls to Action (CTAs).
